# Rational

A decision engine that replaces gut feelings with math. Enter any decision scenario and get analysis powered by 6 mental models:

- **Expected Value** — weighted payoff across all outcomes
- **Base Rate Analysis** — ground-truth probability checks
- **Sunk Cost Detection** — flags when past investment is biasing you
- **Bayesian Confidence** — adjusts your estimates against base rates
- **Survivorship Bias** — warns when you're only seeing winners
- **Kelly Criterion** — optimal allocation sizing (quarter-Kelly for safety)

## Usage

Open `index.html` in any browser. No build step, no dependencies.
